Como se declaran variables en JS?
En Java existen diferentes variables (contenedores para almacenar valores de datos), por ejemplo:
String- almacena texto, conos valores de cadena entre comillas dobles.
int- almacena nmeros enteros.
float- almacena nmeros con decimales.
char- almacena caracteres individuales, como los valores de char entre comillas simples
boolean- almacena valores con dos estados: verdadero o falso.
Para declararlas se debe especificar el tipo y asignarle un valor (el signo igual se utiliza para asignar
valores a la variable)
Se puede escribir de distintas formas dependiendo del tipo de variable, por ejemplo:
Para crear una variable que deba almacenar texto: String name = "John";
System.out.println(name);
Para crear una variable que deba almacenar un nmero: int myNum = 15;
System.out.println(myNum);
Tambin se puede declarar una variable sin asignar el valor y de ese modo asignar el valor ms
tarde: int myNum; myNum = 15; System.out.println(myNum);
Si le asignas un nuevo valor a una variable existente, sobrescribiras el valor anterior, para evitar esto
puedes usar la finalword clave (esto declarar la variable como "final" o "constante", lo que significa
que no se puede modificar y es de solo lectura).

Como se llaman funciones en JS?
Utilizando su nombre seguido de parntesis (). Ya sea utilizando su nombre directamente o
almacenndola en una variable y llamndola a travs de esa variable:
Llamada directa a una funcin declarada: function saludar(nombre) { console.log("Hola, " +
nombre + "!"); }
saludar("Juan"); // Llamada a la funcin
Llamada a una funcin asignada a una variable: var sumar = function(a, b) { return a + b; };
console.log(sumar(3, 5)); // Llamada a la funcin
Llamada a una funcin de flecha (Arrow function): var multiplicar = (a, b) => { return a * b; };
console.log(multiplicar(4, 6)); // Llamada a la funcin
Cmo declaras una funcin interna de un HTML con JS y como declaras un documento
externo con JS?
Para declarar una funcin interna en un archivo HTML utilizando JavaScript, puedes hacerlo dentro
de la etiqueta <script> en el cuerpo del documento o dentro de un atributo onclick, onload, u otro
evento.
En cuanto a la carga de un archivo JavaScript externo en un documento HTML, se utiliza la etiqueta
<script> con el atributo src para especificar la ubicacin del archivo externo. Es importante
asegurarse de que la ruta del archivo JavaScript externo sea correcta y est accesible desde el
archivo HTML para que se cargue y se pueda utilizar.
Cules ciclos de programacin estn disponibles en JS?
Ciclo for: se utiliza cuando conocemos de antemano la cantidad exacta de repeticiones que
deseamos realizar. Tiene tres partes: inicializacin, condicin y expresin de incremento.
Ciclo while: se utiliza cuando queremos repetir un bloque de cdigo mientras se cumpla una
determinada condicin.
Ciclo do-while: se asegura de que el bloque de cdigo se ejecute al menos una vez antes de verificar
la condicin.
Ciclo for-in: para iterar sobre las propiedades de un objeto
Ciclo for-of: para iterar sobre elementos iterables como arreglos o cadenas de texto.
Cules funciones de condicional estn disponibles en JS?
Sirven para controlar el flujo de tu programa y realizar diferentes acciones segn las condiciones que
se cumplan.
Condicional if-else: se utiliza para tomar una decisin basada en una condicin. Si la condicin es
verdadera, se ejecuta un bloque de cdigo, y si es falsa, se ejecuta otro bloque de cdigo.
Condicional switch: se utiliza para realizar mltiples comparaciones y ejecutar diferentes bloques de
cdigo segn el valor de una variable o expresin.
