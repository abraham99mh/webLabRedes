Ejercicio:
Desarrollo de una aplicacin de Red Local
1. Objetivo General
Conocer, investigar e identificar los elementos que se utilizan para configurar una red local y el
acceso a recursos de un equipo en red.
2. Objetivos especficos
Utilizando el programa de Packet Tracer de Cisco, simula las siguientes redes de
computadora, asignando a las computadoras las IP pertenecientes a la red
192.168.0.0., utilizando la ventana de comandos utiliza la funcin ping para verificar la
comunicacin entre cada una de las computadoras. Graba un video de cada red
simulada.
Elabora por cada red una tabla comparativa de cada elemento de red con las
siguientes caractersticas:
